# CueBracket Pro — Phase 3

Adds player management to each tournament:

- Add one player at a time
- Bulk import names separated by lines, commas or semicolons
- Duplicate prevention
- Bracket-capacity enforcement
- Remove players
- Move seeds up/down
- Randomize all players
- Clear the player list
- Capacity progress indicator
- Browser persistence through the existing local-storage tournament service

## Install

Copy these folders into the project root and merge/replace when prompted:

- `app`
- `components`
- `lib`

Then restart development with:

```bash
npm run dev -- --webpack
```
