# CueBracket v0.6.4 patch

- Matches automatically finish and advance when either player reaches the tournament race target.
- Removed the manual Finish Match button and confirmation.

- Completed bracket cards now use the label Finished.

Copy the included `app` and `components` folders into the CueBracket project and replace matching files.
